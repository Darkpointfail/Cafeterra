# Photos du café Cafeterra

Remplacez ces fichiers par vos **5 photos réelles** (même nom de fichier) :

| Fichier | Photo à utiliser |
|---------|------------------|
| `hero.jpg` | Vue panoramique de la salle (grande photo plein écran) |
| `comptoir.jpg` | Comptoir, vitrine et luminaires en rotin |
| `fenetres-briques.jpg` | Salon avec fenêtres et murs de briques |
| `mural-cafe.jpg` | Mur en briques et mural café |
| `salle-profondeur.jpg` | Profondeur de salle, verdure suspendue |

Format recommandé : JPG, 1920px de large minimum pour `hero.jpg`.

Après remplacement, relancez `npm run dev` — aucun changement de code requis.
